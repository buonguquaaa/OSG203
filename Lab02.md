   ‘pwd’ What does this command do?
- Lệnh pwd viết tắt cho Print Working Directory, có nghĩa là in ra các thư mục hiện tại, khi dùng lệnh này nó sẽ trả về đường dẫn đầy đủ của thư mục mà bạn đang làm việc

/ 1.b: Type cd ..

What does this command do?   
How does your prompt change?  

- Lệnh cd (viết tắt của change directory) dùng để di chuyển giữa các thư mục trong hệ thống. Khi gõ cd + tên thư mục, ta sẽ đi từ thư mục hiện tại sang thư mục con tương ứng. Riêng với lệnh ‘cd ..’, hệ thống sẽ đưa ta quay ngược lại thư mục cha của thư mục đang đứng
